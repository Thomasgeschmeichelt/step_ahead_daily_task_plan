# 📋 学生任务表生成工具

## 功能特点

✨ **主要功能**
- 📝 可视化表单输入
- 👁️ 实时预览效果
- 📥 导出高清PNG图片（3倍分辨率）
- ➕ 动态添加/删除任务和计划
- 🎨 美观的界面设计

## 使用说明

### 1. 填写任务信息

**当日任务表部分：**
- 选择日期
- 填写到/离校时间
- 填写今日课程
- 输入学生姓名
- 添加或编辑任务列表（规定时间、学习安排、实际时间、完成情况）

**次日学习计划部分：**
- 选择日期
- 输入学生姓名
- 添加或编辑计划列表（任务内容、规定时长、备注）

### 2. 状态类型说明

- **完成**：绿色显示，带✓
- **超时完成**：红色加粗显示，带✓
- **未完成**：红色加粗显示
- **午休**：特殊橙色背景，跨4列显示
- **无**：普通黑色文字

### 3. 预览和导出

- 点击 **🔄 更新预览** 按钮刷新右侧预览
- 点击 **📥 导出PNG图片** 导出高清图片
- 导出的图片会自动命名为：`学生姓名_日期任务表.png`

## 技术特点

- 📱 响应式设计，适配不同屏幕
- 🎨 渐变背景和现代UI设计
- 🖼️ 使用html2canvas生成高清图片（3倍分辨率）
- ⚡ 纯前端实现，无需服务器

## 浏览器要求

- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 支持ES6+语法
